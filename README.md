## Overview

This is my **personal developer portfolio website** — designed to present my skills, projects, education, and certifications in a clean and professional manner. The site is fully responsive across all devices and features smooth animations for an engaging user experience.

## Features

-  **Smooth Animations** — Scroll-triggered and entrance animations for a polished feel
-  **Fully Responsive** — Optimized for desktops, tablets, and mobile devices
-  **Clean UI Design** — Minimal and professional layout with intuitive navigation
-  **Project Showcase** — Live demo and source code links for each project
-  **Sticky Navbar** — Easy navigation across all sections
-  **Filterable Projects** — Filter projects by category (Full Stack, Frontend, UI)
-  **Resume Download** — Direct link to download/view resume
-  **Contact Section** — Quick access to email, LinkedIn, and GitHub

## Getting Started

To run this project locally:

```bash
# 1. Clone the repository
git clone https://github.com/vinothinivijayakumar-dev/portfolio-website.git

# 2. Navigate to the project folder
cd portfolio-website

# 3. Open in browser
# Simply open index.html in your preferred browser
```

> No build tools or dependencies required — it's plain HTML, CSS, and JavaScript!
